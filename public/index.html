Image Processing API

Welcome

Description:

_ Image Processing API is an application where the user enter some parameters [image name , desired width & height] in the url to resize an image in our database.

_ Please provide just positive integers as you type the width & height to generate the thumbnail from the available images.

_ Available images are : [aerial_view, boat_on_lake, galaxy, sunset]

_ End points (example):

/api/images?image=aerial_view

/api/images?image=aerial_view&width=[number]&height=[number]
